# Telemetron Quick‑Start ⚡️

This guide walks you through the fastest way to feed your **existing REST API** into Telemetron so that agents can see live data for every customer.

> **Prerequisites**  
> • A valid **OpenAPI 3.x spec** of your public API (JSON or YAML)  
> • Your organization’s **`TELEMETRON_API_KEY`** (create one in *Console → Settings → API Keys*)

---

## 1 · Open the *Import from OpenAPI* wizard

1. Log in to the [Telemetron Console](https://console.telemetron.com).  
2. Go to **Connections** in the left sidebar.  
3. Click the **Import from OpenAPI** tab.
